Denna webbapplikation är utvecklad med Blazor-ramverket och använder Syncfusion. På denna webbsida kan man lagra information om olika personer, 
såsom för- och efternamn, adress, telefonnummer, plats med mera. Det ska även vara möjligt att se var varje person bor med hjälp av en karta.
För närvarande har jag inte implementerat kartfunktionen, då min Syncfusion API inte längre är tillgänglig. 
Jag behöver använda ett API från Google, som är kopplat till Google Maps, för att kunna visa platserna för olika personer. Detta API är dock inte gratis och medför kostnader
